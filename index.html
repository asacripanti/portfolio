Edit your own MTG deck

Build and plan out your own Magic the gathering deck. This app was built with HTML, CSS, JS, Python, PostgreSQL. Look up any card from the Magic universe just by typing in the search bar, the website then uses the Magic API to retrieve the results from the database. All decks, cards and usernames are stored in PostgreSQL database so you can come back at anytime to add or remove cards.
